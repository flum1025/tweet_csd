Tweet_CSD
===========

##What is it?

[CSD](http://github.com/flum.pw/csd)のデータをツイートするやつ。  

動作確認はruby 2.0.0p481とruby 1.9.3p484です。

##How to Use
環境に合わせて
pc_name,
@reqest_url,
reply_id,
consumer_key,
consumer_secret,
access_token,
access_token_secret
の変数を指定してください。

cronに登録しておくと自動でツイートできます。  
  

質問等ありましたらTwitter:[@flum_](https://twitter.com/flum_)までお願いします。
